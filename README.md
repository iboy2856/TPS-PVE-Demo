# Unity 3D PVE 射击游戏求职项目

本项目用于展示第三人称射击游戏的基础开发能力，包含角色控制、相机跟随、动画切换等核心功能。

### 已完成功能
- 玩家角色移动（基于胶囊体和 PlayerCtrller 脚本）
- 第三人称相机跟随与鼠标旋转（带平滑和距离限制）
- 基础动画状态机（待机/移动通过 Speed 参数切换）

### 开发环境
- Unity 版本：2021.3.x（或你实际使用的版本）
- 语言：C#

### 如何运行
1. 克隆本仓库
2. 用 Unity Hub 打开项目文件夹
3. 打开 `SampleScene` 并点击运行
